# Memphis Roots

A community cultural-memory map for The Big We, part of the Restorative Development work anchored on Clayborn Temple in Memphis.

**Live:** https://judy-huynh.github.io/restodev/

The site is in `built-site/` and deploys to GitHub Pages:

- `index.html`: the Clayborn Time Machine landing page
- `app.html`: the living map
- `layers.html`: the Memphis city-data base map

Run locally: `cd built-site && python3 -m http.server 8000`

Built with Judy Huynh (Rebuild by Design).
